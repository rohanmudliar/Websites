Weather App

Hover over to know more.

A Weather Web App buid using Php and mySQL. It shows the weather of the entered city or state in a div block. It is mobile responsive as it is built on a bootstrap layout. This webpage fetches data form another website and crops the required data using 'explore' function.

It also displays an error if invalid city name or state name is entered. If the link or the HTML skeleton of the source page is changed, we have to modify the code in the HTML as well. As an alternative with this method, we can use Weather API.
